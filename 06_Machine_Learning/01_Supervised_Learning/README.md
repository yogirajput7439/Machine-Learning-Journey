# 📚 What is Supervised Learning?

Supervised Learning is a type of Machine Learning where models learn from **labeled data**.

### Simple Example:
| Input | Output |
|------|------|
| House Features | House Price |
| Student Study Hours | Exam Score |
| Customer Data | Fraud / Not Fraud |

The model learns patterns from existing data and predicts outputs for new data.
