In this journey, I am learning how machines discover hidden patterns and structures from data **without labeled outputs**.

---

# 📚 What is Unsupervised Learning?

Unsupervised Learning is a type of Machine Learning where the model works on **unlabeled data** and tries to identify:
- Hidden patterns
- Similar groups
- Data structures
- Relationships between data points
